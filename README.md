Something Broke IDK is an example project of a way to help support engineers get more information from customers regarding their query. Sometimes customers can be less discriptive when it comes to what they are going through and that can make it difficult to move forward. 

This website helps 
- figure out what made them write in 
- what time it happened (to track down logs)
- how urgent the issue is
- if it is blocking any necessary features or impeding work

You can view the website here: https://pinkpushpop.github.io/somethingbrokeidk
